## Overview

This project focuses on developing an **intelligent virtual teaching system** that enables users to program and train an industrial robot arm within an **AR/VR environment**. By leveraging **digital twin technology**, the system allows intuitive robot teaching while maintaining **sub-millimeter trajectory accuracy**.

---

## Challenge

Traditional robot teaching methods are time-consuming and require on-site operation using teach pendants. The main challenges addressed in this project were:

- Enabling **precise robot arm teaching** in a virtual environment  
- Maintaining **< 1 mm trajectory accuracy** between virtual and real robots  
- Providing an intuitive human–robot interaction interface  
- Ensuring reliable transfer of virtual paths to real-world execution

---

## Core Technologies

- Augmented Reality (AR) and Virtual Reality (VR)  
- **Digital Twin** of an industrial robot arm  
- Multi-axis motion control  
- HTC Vive for spatial tracking and user interaction  
- Robot **forward and inverse kinematics**  
- Virtual teaching and trajectory recording  
- Offline path planning

---

## Contributions

- Implemented **forward and inverse kinematics** to map user inputs to robot motion accurately  
- Developed a **digital twin system** to synchronize virtual and physical robot behavior  
- Achieved **sub-millimeter (<1 mm) trajectory accuracy**  
- Validated the system on precision applications such as **spray painting**

---

## Results & Impact

- Reduced robot programming time compared to traditional teach pendant methods  
- Improved teaching precision and repeatability  
- Demonstrated the feasibility of **AR-based robot teaching** for industrial applications  
- Provided a scalable framework for future smart manufacturing and robotic automation systems
