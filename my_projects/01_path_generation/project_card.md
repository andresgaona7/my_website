### Robot Arm Automatic Path Generation Technology (2019–2021)

**Challenge**  
Automate optimal path planning and reduce manual robot programming time.

**Solution**  
Developed a **point cloud–based path generation system** that automatically produces **MCCL-compatible tasks programs** with integrated G-code motion execution.

**Core Tech**  
Point Cloud Processing · Path Planning ·  Motion Control · G-Code

**Impact**  
Reduced program generation time to **under 1 hour** and delivered **~9.6M NTD productivity impact**.
